## Prerequisites
- [python3.13](https://www.python.org/downloads/release/python-3130/)
- [pip](https://pypi.org/project/pip/)

## Installation
Launch the `make install` command to setup the repository.

## Usage

### Using the CLI (Recommended)
The project includes an interactive CLI for running solutions:

```bash
# Interactive mode - select day and part interactively
poetry run aoc

# Direct mode - specify day and part
poetry run aoc --day 1 --part 1
```

### Direct Python execution
Alternatively, you can run solutions directly:
```bash
poetry run python advent_of_code_2025/day_XX/part_YY.py
```

## Cleanup
Launch the `make uninstall` command to cleanup the repository.
